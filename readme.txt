About the app:
Name: Attendance System
Author: GENESSA LIASO
Github Url: https://github.com/genessa1990/attendance
Description: Basic attendance system where user can add students, add block and take attendance per block. 
	     User is also able to view attendance of students per block per dates when attendance was taken.



Requirements:
*working xampp
*run apache
*run mysql
*need internet for bootstrap


How to setup 
First create database named attendance_db in phpmyadmin
then import attendance_db.sql in folder
After copy the folder name attendance within SourceCode and paste in htdocs in xampp
go to localhost/attendace/public to use system
